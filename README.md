# Vaultmont-Lab

 
 
## Getting Started

1. Clone from Github 
```
git clone https://github.com/themewagon/Crypgo.git](https://github.com/devosumba/Vaultmont-Lab
```
2. Install Dependencies
```
npm i
```
3. Run local server
```
npm run dev
```
